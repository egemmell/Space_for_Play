# Space_for_Play
Calculated formal space for play indicators

## Data Sources
- **CanMap Content Suite 2020 v.3** — available from DMTI Spatial via 
  institutional licence. Place shapefiles in `Data/CMAparks/` and 
  `Data/CanMapPC/`.
- **OpenStreetMap** — downloaded automatically by the script via the 
  `osmdata` package.

## Data folder structure
Data/
├── CMAparks/
├── CanMapPC/
└── Space_for_Play_Dimension/
